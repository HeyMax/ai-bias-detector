# 🔍 AI 偏见检测器 (AI Bias Detector)

> 别盲信 AI 推荐。

**AI Bias Detector** 是一个轻量级、零配置的 Chrome 浏览器插件，能够实时检测 AI 聊天机器人推荐中的潜在偏见——就在你聊天的地方。

完全本地运行。不需要 API Key。没有后端服务。你的数据不会离开浏览器。

## 为什么需要它？

当你问 ChatGPT *"推荐一个项目管理工具"*，它可能会推荐 Notion——但你知道 Notion 是 OpenAI 的合作伙伴吗？当你问 Gemini，它可能偏向推荐 Google Workspace——这是它自家的产品。

AI 助手非常有用，但它们的推荐可能受到以下因素影响：

- **商业关系** — AI 厂商与被推荐品牌之间的合作/投资关系
- **训练数据偏见** — 热门产品在训练数据中有更多曝光
- **修辞偏向** — 使用"最好的"、"唯一的"等绝对化表述，不提替代方案
- **信息缺失** — 不提价格、不说缺点、不列竞品

AI Bias Detector 让这些隐形的模式变得可见。

## 功能特性

### 🎯 零配置偏见检测
安装即用。插件会自动分析支持平台上的 AI 回复。

### 🏢 利益冲突数据库
一个策划维护的、开源的 AI 厂商与品牌商业关系数据库（OpenAI ↔ 微软, Google ↔ Workspace, Anthropic ↔ 亚马逊等）。检测到冲突时会即时提醒。

### 📊 偏见指数 (0-100)
每条 AI 回复都会得到一个综合评分，基于：
- **集中度** — 是只推荐了一个选项，还是给了平衡的多个选择？
- **修辞** — 是否使用了绝对化语言（"最好的"、"唯一的"）？
- **信息缺失** — 是否遗漏了缺点、价格或替代方案？
- **来源不透明** — 是否引用了模糊的"研究"而没有具体来源？

### 💬 一键追问
基于检测到的偏见，自动生成智能追问提示。点击即可直接发送到聊天中——无需复制粘贴。

### 🌍 双语支持（English / 中文）
完整支持中英文界面和检测规则。

### 👤 两种模式
| 模式 | 适合谁 | 看到什么 |
|------|--------|---------|
| **简洁模式** | 所有人 | 清晰的信号 + 建议 |
| **专家模式** | 开发者和研究者 | 完整证据 + 原始规则 + 自定义规则 |

## 安装

### 从 Chrome 应用商店安装（即将上线）

> 我们正在准备首次发布。给这个仓库点个 Star 以获取通知！

### 从源码安装（开发者模式）

```bash
# 克隆仓库
git clone https://github.com/chenzhaohao/ai-bias-detector.git
cd ai-bias-detector

# 安装依赖
npm install

# 构建扩展
npm run build

# 构建产物在 dist/ 目录下
```

然后在 Chrome 中加载：

1. 打开 `chrome://extensions/`
2. 开启右上角的**开发者模式**
3. 点击**加载已解压的扩展程序**
4. 选择 `dist/` 目录

## 设计理念

1. **隐私第一** — 所有分析在本地运行。我们永远不会收集、传输或存储你的对话。
2. **数据开放** — 利益冲突数据库完全透明，由社区维护。
3. **不做判断** — 我们不告诉你应该怎么想。我们呈现信号，由你自己决定。
4. **包容设计** — 简洁模式给所有人，专家模式给进阶用户。
